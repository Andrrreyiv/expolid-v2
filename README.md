# ЭкспоЛид

**Захват контактов на выставках с AI-распознаванием визиток, расшифровкой голоса и автогенерацией follow-up'ов.**

🌐 Демо: [https://dist-bmicbecs.devinapps.com/](https://dist-bmicbecs.devinapps.com/)
🔧 Backend: [https://expolid-backend-hhpanzkc.fly.dev/api/health](https://expolid-backend-hhpanzkc.fly.dev/api/health)

---

## Что приложение умеет

### Захват контакта за 20 секунд
- **Фото визитки** → AI (Google Gemini) распознаёт ВСЕ поля: имя, компания, должность, телефон, email, сайт, Telegram, WhatsApp, LinkedIn — даже рукописные дописки
- **QR-код визитки** (vCard / MECARD) — парсится прямо в браузере без AI
- **Голосовая заметка** → AI (Groq Whisper) расшифровывает в текст
- **Заметка вручную** — для уточнений
- **Фото человека** (опционально) — чтобы запомнить, с кем разговаривали
- **Павильон / стенд** — где сделан контакт

### AI-помощь после контакта
- **Резюме встречи** — 1-2 предложения сути из голоса+заметок
- **Договорённости** — что обещали друг другу
- **Следующий шаг** — конкретное действие + срок (создаётся авто-задача)
- **Скоринг 1-100** + причина (горячий/тёплый/холодный)
- **Follow-up письмо** в 4 сценариях: вводное письмо / коммерческое предложение / приглашение на встречу / план звонка
  - С персонализацией: «скидка 15%», «шоу-рум», «кейс из вашей отрасли» — вписывается в текст естественно

### Многопользовательский режим
- Команда: владелец, менеджер, стендист
- Назначение контактов на менеджера
- Дашборд KPI: всего контактов / горячих / просрочено / среднее время до follow-up
- Real-time уведомления в команде через SSE («Иван добавил контакт Acme Corp»)

### Мульти-контакт
- Уникальная фича: «визитка Алексея, разговаривал с его коллегой Иваном» — сохраняются оба
- Конкуренты так не умеют

### Offline-first
- IndexedDB через Dexie — контакты сохраняются локально, если Wi-Fi на стенде упал
- Авто-синхронизация при появлении сети
- Индикатор сверху: «В очереди: N» / «Отправлено: N»

### PWA
- «Добавить на главный экран» в браузере — работает как нативное приложение
- Push-уведомления через Web Push API (просроченные задачи + новые контакты команды)

### Экспорт
- Excel со всеми 24 колонками: дата, выставка, ФИО, компания, должность, телефон, email, сайт, Telegram, WhatsApp, LinkedIn, тип, статус, AI-балл, резюме, договорённости, следующий шаг, дата напоминания, связь визитка↔контакт, поговорил с, менеджер, павильон, стенд, источник

### Интеграции
- **Telegram-бот** (опционально): `/start <код>` — отправь фото визитки и голосовое в бот → контакт создан в приложении
- **amoCRM** (опционально): подключить долгосрочный токен → push контакта одним кликом → создаётся сделка + контакт в CRM

## Как пользоваться

### 1. Регистрация
1. Открыть [https://dist-bmicbecs.devinapps.com/](https://dist-bmicbecs.devinapps.com/)
2. Нажать «Регистрация»
3. Ввести имя, email, пароль (минимум 6 символов)
4. Поле «Компания» — опционально (можно заполнить позже в Настройках)
5. Браузер предложит сохранить пароль — соглашайтесь, на других устройствах подставится автоматически

### 2. Создать выставку
1. Настройки → вкладка **Выставки** → «+ Добавить выставку»
2. Название, город, площадка, даты
3. Первая выставка автоматически становится активной — все новые контакты привязываются к ней

### 3. Захват контакта
**На главной:**
- **«Записать контакт»** — открывает 2-шаговый capture: фото визитки + голос + заметка → шаг 2: фото человека (опц.) + тип/статус → AI-обработка
- **«Быстрая голосовая заметка»** — записать только голос, контакт создаётся сразу

**На стенде с медленным Wi-Fi**: всё сохраняется в IndexedDB, синхронизация автоматически при появлении сети.

### 4. Работа с контактом
- Контакты → выбрать → редактирование инлайн через карандаш
- Создать follow-up: «Сгенерировать письмо» → 4 кнопки сценариев + поле персонализации → AI пишет за 5 сек
- Создать задачу со сроком (или авто-создаётся из «следующего шага» AI)
- Поделиться с менеджером (assign)

### 5. Команда
Настройки → **Команда** → «+ Добавить» → email + пароль → роль:
- **Owner** — всё, включая удаление компании
- **Manager** — переименование компании, управление командой, все контакты
- **Staff** — только свои контакты

### 6. Шаблоны follow-up
Настройки → **Шаблоны** → создать шаблон по типу (intro/proposal/invite/call) → AI будет адаптировать его под каждый контакт.

### 7. Экспорт
Нижняя навигация → **Экспорт** → Excel загружается мгновенно.

### 8. Push-уведомления
Настройки → **Push** → «Включить уведомления» → «Отправить тест» (проверка) → дальше уведомления приходят автоматически:
- Когда коллега добавляет контакт
- Когда у вас просроченная задача

### 9. Multi-device
- Логин/пароль работают на любом устройстве — заходите с компьютера, телефона, планшета параллельно
- Чекбокс «Запомнить меня (30 дней)» — токен живёт месяц вместо недели
- Менеджер паролей сохранит креды → на любом другом устройстве введёт за вас

## Технологии

| Слой | Стек |
|---|---|
| Frontend | React 18 + TypeScript + Vite + Tailwind + shadcn/ui + Dexie (IndexedDB) + PWA |
| Backend | FastAPI + SQLAlchemy 2 + Pydantic v2 + SQLite (dev) / PostgreSQL (prod) |
| AI | Google Gemini 2.5-flash (vision+chat) · Groq Whisper-large-v3-turbo (STT) · OpenAI (опц. fallback) |
| Auth | JWT (access token, 7/30 дней) + bcrypt |
| Real-time | SSE (`/api/events/stream`) |
| Push | Web Push API + VAPID |
| Деплой | Frontend: статика на любой CDN/Devinapps · Backend: Fly.io (с persistent volume для SQLite) |
